# project title API DashBoard

Public API Explorer Dashboard: Build a multi-panel dashboard that fetches data from 
two public REST APIs simultaneously using Promise.all. Display loading states, 
handle and display errors gracefully, allow the user to trigger re-fetches, and format the 
results cleanly in the DOM. All async logic must use async/await. Use ES6 modules 
by splitting the code into at least two .js files.
