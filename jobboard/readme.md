# project title --> Job Board Application:
 Build a job board where companies can post jobs and 
applicants can apply. Use ModelForm for job posting and application submission with 
full validation, CreateView and ListView as class-based views, Django 
authentication for company login, LoginRequiredMixin to protect posting, and 
select_related on the application list to avoid N+1 queries. An admin panel must 
show annotated counts of applications per job.
